# PDF Local Work

Local Next.js helper for fast PDF merge and split workflows on `http://localhost:3002`.

## Commands

```bash
npm install
npm run dev
```

## v1 features

- Browse folders by local path through the server-backed API
- Optional browser directory picker workflow in supported browsers
- Multi-select PDFs and images, then merge into a single PDF
- Split one PDF by ranges or into one PDF per page
- Write output into the same folder with collision-safe naming
